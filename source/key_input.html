Virus Link Scanner

Thanks for installing.
Please enter your Metascan Online API key to begin using this extension.

How to use

  • While on a web page, select and right click on a link that your interested in downloading but are unsure if the link is safe
  • In the context menu, select the following option : "Scan link for viruses"
  • The extension will begin doing the analysis and determine the appropiate steps
  • If a virus exists, you will be taken to a new page with the results of the scan. No download will occur
  • If the file is clean, then you will be prompted to save the file instead

Dont have a Metascan Online API key?

If you need a Metascan Online API key, you can get a key from the OPSWAT Portal.

  • Navigate in any web browser to https://portal.opswat.com
  • If you have not already, create a free account
  • After logging in, navigate to the Security Applications tab
  • Click on the Metascan Online API tab to activate your Metascan Online API key
